About

This was built as my final project for MIT's legendary class, "How To Make (almost) Anything." The idea of this project is to create a relaxing and automated table that resembles a Japanese zen garden. The design is inspired by a Kickstarter project that came out a few years ago. Visit my class website to read a more step-by-step description of my design process.
